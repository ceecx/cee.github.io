2019-10-08

颁发 SSL 证书无响应或超时，出现报错： 283 Failed to obtain SSL for domain. [issueSSLForDomain] [createVirtualHost] 80％ 是 DNS 问题，可能你用了反代或主站用了 CDN，导致获取不到正确的 Server IP 而颁发失败，我的解决办法是：解析到正确的 Server IP 生效后，SSH 执行以下代码然后面板内再次申请签发证书：

wget -O - https://get.acme.sh | sh

而 20％ 是被 Lets Encrypt 限制，说明近期同一域名颁发次数过多，Lets Encrypt 限制每周同一域名签发次数是 5 次（不知道是否记错，反正次数不多）。
